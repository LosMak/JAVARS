# iOS Калькулятор

Красивый веб-калькулятор в стиле iOS с полной функциональностью и исправленными ошибками.

## Функции

- ✅ Базовые арифметические операции (+, -, ×, ÷)
- ✅ Операции с процентами (%) - исправлено
- ✅ Изменение знака числа (±) - исправлено
- ✅ Поддержка десятичных чисел - исправлено
- ✅ Очистка дисплея (C) - исправлено
- ✅ Красивая кнопка равно (=) - исправлено
- ✅ Красивый дизайн в стиле iOS
- ✅ Адаптивная верстка для всех устройств
- ✅ Поддержка клавиатуры
- ✅ Вибрация на мобильных устройствах
- ✅ Правильная обработка ошибок (деление на ноль)
- ✅ Обработка очень больших и малых чисел
- ✅ Цепочки вычислений работают корректно

## Исправленные ошибки

### ✅ Критические исправления

- **Кнопка равно** теперь корректно отображается и работает
- **Деление на ноль** показывает "Ошибка" вместо зависания
- **Дробные числа** корректно обрабатываются в цепочках операций
- **Проценты** правильно вычисляют значения вроде "100 + 2% = 102"

### ✅ Улучшения функциональности

- **Смена знака** работает в любых контекстах
- **Большие числа** корректно форматируются (до 999999999999)
- **Цепочки операций** выполняются последовательно
- **Отрицательные числа** правильно отображаются
- **Округление** работает точно до 8 знаков после запятой

## Запуск

1. Откройте терминал в директории проекта
2. Запустите локальный сервер:

   ```bash
   python -m http.server 8080
   ```

3. Откройте браузер и перейдите по адресу: `http://localhost:8080`

## Управление

### Кнопки

- **Цифры 0-9** - ввод чисел
- **+, -, ×, ÷** - арифметические операции
- **=** - вычислить результат (кнопка внизу)
- **C** - очистить калькулятор
- **±** - изменить знак числа
- **%** - вычислить процент
- **.** - десятичная точка

### Клавиатура

- Цифры и операторы работают с клавиатуры
- **Enter** или **=** - вычислить
- **Escape** или **C** - очистить
